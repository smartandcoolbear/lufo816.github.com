折腾了一晚上，最后竟然用这么简单的方法解决了。

安装
sudo add-apt-repository ppa:nilarimogard/webupd8
sudo apt-get update
sudo apt-get install ap-hotspot

配置
sudo ap-hotspot configure

启动
sudo ap-hotspot start

停止
sudo ap-hotspot stop
